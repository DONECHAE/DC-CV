**기술:** Python, FastAPI, PostgreSQL, LLM

- 운영 자동화 및 검증 로직 구현

### 성과
- 주석 기반 재무정보 추출 정확도 90%대  
- 파이프라인 자동 운영 성공률 99% 이상
