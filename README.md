    %% DB

    D2[재무 · 거시 · 공시 · 주석 스키마]

    E1[FastAPI 내부 API]

    %% 데이터 흐름

    A2 --> B1
    A3 --> B1
    A4 --> C1

    D1 --> E1

    E2 --> C1
    E3 --> D1
